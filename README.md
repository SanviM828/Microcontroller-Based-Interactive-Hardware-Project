This repository documents an ongoing self-project focused on developing an **interactive embedded system prototype**.  
The project explores the integration of **synchronized LED light patterns, audio effects, and motor-driven actuation** to simulate dynamic real-time behavior.

Originally inspired by interactive prop (Flame Chompers) from entertainment media (Arcane series), this project aims to reimagine such concepts through an engineering and embedded systems perspective.

---

## Project Overview
- **Objective:** To design and prototype an interactive hardware system demonstrating real-time synchronization of light, sound, and motion.  
- **Platform:** Arduino Nano (primary microcontroller)  
- **Key Features (in progress):**  
  - Dynamic LED blinking patterns with increasing frequency.  
  - Audio playback synchronized with LED sequences.  
  - Motor-driven actuation for mechanical motion effects.  
  - Compact integration of electronics into a custom mechanical design.  

---

## Skills & Tools
- **Embedded Programming:** Arduino IDE (C/C++)
- **Circuit Simulation**: Simulink, LTSpice
- **Hardware Development:** Circuit prototyping, LED arrays, sound modules, motors  
- **Design Tools:** CAD modeling for housing/mechanical parts (Fusion360 or SolidWorks)  
- **Other Tools:** Soldering, perfboard integration, 3D printing (planned)
